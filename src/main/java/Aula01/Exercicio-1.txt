Dentre as vantagens de se utilizar a linguagem de programação Java, explique o porquê do Java ser considerada uma tecnologia portável?

Pois um mesmo código pode ser interpretado em diversas plataformas com um mesmo código. Onde a plataforma Java compila o código em bytecodes e joga para JVM(Java Virtual Machine) e depois interpreta para o Sistema Operacional.

Devido a isso: 
°Não existe incompatibilidade entre os tipos de dados;
°Reduz custo;
°Uma aplicação pode ser desenvolvida e testada uma única vez que ela roda em diversas plataformas sem a necessidade de alterações.
